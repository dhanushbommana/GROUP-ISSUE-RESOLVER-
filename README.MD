# 💬 WhatsApp Group Management System

A lightweight web application for managing WhatsApp group requests with Google OAuth login, email notifications, and admin dashboard.

## ✨ Features

- 🔐 **Google OAuth 2.0 Login** - Secure authentication
- 👤 **User Profiles** - College details management
- 📝 **Request System** - Submit and track requests
- 📧 **Email Notifications** - Auto alerts for requests and responses
- 👑 **Admin Dashboard** - Manage all requests
- 📊 **Statistics** - Real-time request tracking
- 🎨 **Responsive Design** - Works on all devices

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite3
- **Auth:** Google OAuth 2.0
- **Email:** Gmail SMTP
- **Frontend:** HTML5, CSS3, Bootstrap 5
